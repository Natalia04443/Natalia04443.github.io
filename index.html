Reglas del juego

El juego de las N reinas tiene una regla principal: colocar N reinas en un tablero de ajedrez de N x N de forma que ninguna reina pueda atacar a otra.

  • Tablero: N x N
  • Regla: No compartir fila, columna ni diagonal
  • Objetivo: Colocar N reinas correctamente
  • Ejemplo: Tablero de 8x8 requiere 8 reinas
